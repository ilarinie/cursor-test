# Placeholder Image Downloader

A simple TypeScript Node.js application that downloads a 600x400 placeholder image from placehold.co.

## Installation

```bash
npm install
```

## Usage

### Development (using ts-node)
```bash
npm run dev
```

### Production (build then run)
```bash
npm run build
npm start
```

The image will be saved as `placeholder-600x400.png` in the project root directory.
